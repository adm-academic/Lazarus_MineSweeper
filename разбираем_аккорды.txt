Разбираем аккорды! То есть функционал нажатия на две клавиши мыши одновременно.

Когда мы АКТИВИРУЕМ аккорд, то имеем для каждой ячейки должны знать:
 1- количество соседей для ячейки... КС [+]  function get_neighbors_count
 2- количество ячеек аккорда... КА [+] function chrord_get_cells_count
 3- ( подсчитать количество флагов в аккорде КФА ) [+] function chord_get_flags_count

 4- открытие аккорда просчитывается только при отпускании клавиш, а не при нажатии [+]

 5- если КС=КФА, то открываем все ячейки аккорда, не помеченые флагами.
   Открытие ячеек должно быть рекурсивным и при этом уметь подрывать бомбы
   при открытии, а также отрывать рекурсивно прстранства с пустыми клетками [+]

 6- при перемещении по полю курсора по ячейкам с зажатым аккордом должно отрабатывать:
   6.1- показ аккорда [+]
   6.2- открытие аккорда [+]
   6.3- при перемещении аккорда по гриду не должно быть выходов за пределы размерностей [+]

 7- при попытке взять аккорд на закрытой клетке она не должна открываться или помечаться флагом [+]
