Сейчас у меня аккорд берётся без флагов, а должен с флагами!

Когда мы АКТИВИРУЕМ аккорд, то имеем для каждой ячейки должны знать:
 - количество соседей для ячейки... КС [+]  function get_neighbors_count
 - количество ячеек аккорда... КА [+] function chrord_get_cells_count
 - ( подсчитать количество флагов в аккорде КФА ) [+] function chord_get_flags_count

 - открытие аккорда просчитывается только при отпускании клавиш, а не при нажатии [+]

 - если КС=КФА, то открываем все ячейки аккорда, не помеченые флагами.
   Открытие ячеек должно быть рекурсивным и при этом уметь подрывать бомбы
   при открытии, а также отрывать рекурсивно прстранства с пустыми клетками [+]

 - при перемещении по полю курсора по ячейкам с зажатым аккордом должно отрабатывать:
   - показ аккорда [+]
   - открытие аккорда [+]
   - при перемещении аккорда по гриду не должно быть выходов за пределы размерностей [+]

 - при попытке взять аккорд на закрытой клетке она не должна открываться или помечаться флагом,
   может быть стоит перенести логику щелчков мыши из MousePress в MouseUp ??? [-]
